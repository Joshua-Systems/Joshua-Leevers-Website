Rocketry Club Ground Systems Lead/Avionics dev

Date: june 2025-Present

Working on the ground systems for the rocketry club, including avionics development and testing.

Currently working on the ground station software and hardware integration.

So far I've designed the ground station hardware and I've started working on the peripherals like the voltage regulators and the firmware for the ESP32 MCU we are using for the physical layer stand in with a Hope RF LoRa chip.

I'm also leading the development of the network stack to send live telemetry data to a command layer and the rocketry club website using a natively created netowrk stack to process either LoRa or FSK packets that can deal with either in real time

Our next goals are to get the telemetry system set up for the AURC and the NZRA on a level 2 build.
