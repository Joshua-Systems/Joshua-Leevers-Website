Project Portfolio

ESP32 LoRa Driver

Developed a driver for ESP32 LoRa communication.

Implemented schemes for transmission over LoRa protocol.

SDR (HackRF/RTL-SDR) Tx/Rx Graphs in GNU radio

Developed and tested GNU Radio Graphs for the duplex telemetry of the ground station.

This development had succeeded and we were able to send test messages round loop.

The system still needs optimisation the working product is currently being

tested on a duplex set of Raspberry Pi's

Endurostack Network Stack

Currently developing a recreation of the Endurosat Network Stack in Python

Currently I've developed the TP network layer and aided in developing the Datalink layer.

RCA -> Balanced Stereo Audio Mixer PCB

Designed, ordered, and soldered together a PCB which performs impedance matching

and balancing of signal voltages. This design improved audio quality and was tested

on an RCA connection speaker and is being used in the field currently by an audio company in West Auckland.

that I designed it for and sold it to.
